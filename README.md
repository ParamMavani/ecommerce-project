# 🛒 Ecommerce Project

A simple ecommerce web application with cart system and multiple payment options.

---

## ✨ Features

- Login & Signup
- Product Listing
- Add to Cart
- Cart Management (Add/Remove items)
- Checkout System
- Payment Options:
  - PayPal
  - UPI
  - Cash on Delivery (COD)
  - Card (Demo)

---

## 🧰 Tech Stack

- HTML, CSS, JavaScript  
- PHP  
- MySQL  

---

## ⚙️ Setup

1. Install XAMPP  
2. Move project to `htdocs/ecommerce-project`  
3. Start Apache & MySQL  
4. Open: `http://localhost/ecommerce-project`
